## `overview`

> Software developer working across full-stack, backend, and ML/MLOps + generative AI. Lately I care less about shipping features and more about how the *system* underneath them is put together — data flow, failure modes, the boring plumbing nobody sees. Freelancing on projects where I get to own that whole shape, not just a component of it. I build clones of well-established platforms as a way of reverse-engineering how serious systems are actually built, and I write about what I find along the way.
